Spantus - speech analitical tool
Copyright (C) 2007-2009 Mindaugas Greibus <spantus@gmail.com>

Spantus calculates and depicts features of speech signal. Spantus can analyse wav files or stream from micraphone.

1. Status of the project

 Spantus is currently in initial development phase.

2. Start program

Windows: click on spantus.exe
Linux: execute spantus.sh

3.  Build Spantus
 
3.1 Source code: http://sourceforge.net/projects/spantus/

3.2 Build instructions: http://spantus.wiki.sourceforge.net/DeveloperNotes
